How It Works

Take your regular list-menu, like this:

<ul class='menu'>

<li>Home</li>
<li>About</li>
<li>Generic Menu Item</li>
<li>Contact</li>

</ul>
Next...

Having called Flex.js, simply place this line in your js file:

$('.menu').flex({});

There are plenty of options to customise your menu as you like:

  • Colours, fonts etc.
  • Direction
  • Responsiveness
  • Spacing
  • Order
Colours and Fonts

Good news! Flex is as code-agnostic as possible, which means your menu items will inherit styles like these from the CSS you set and it won't interrupt what flex is doing!

Direction

In order to change the direction of the menu, find this part of the script:

direction: 'row'

You can change the option here to either row (horizontal), or column (vertical).

Responsiveness

Flex will automatically change direction from a row to a column when the screen becomes smaller than 720 pixels. To change this, find this code section:

if($(window).width() < 720){
$('.flex-grid-ross-css').css({
'flex-direction':'column'
});
}

$(window).resize(function(){
if($(window).width() < 720){
$('.flex-grid-ross-css').css({
'flex-direction':'column'
});
}

else {
$('.flex-grid-ross-css').css({
'flex-direction':'row'
});
}

You can remove this completely, or change it to a different screen width. You can also swap column for row.

Space Between

In order to define the space around your menu items, first, find the options at the top, and set the following option:

justify: 'space-around'

You can change the option to any of the following:

  • Flex-start - Aligns the items to the left-hand side
  • Flex-end - Aligns the items on the right
  • Center - Justifies the items into the center
  • Space-Between - Leaves space between each item, but not at the edges
  • Space-Around - Similar to Space-Between, but also puts space at the edges of the menu.

Order

This section of code selects each item in your list and applies an order to them. To change the order, simply change the number in the qoutes. The lower the number, the earlier the order.

$(one).css({
'order':'1'
});

$(two).css({
'order':'2'
});

$(three).css({
'order':'3'
});
